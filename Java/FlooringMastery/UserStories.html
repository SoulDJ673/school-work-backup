User Stories


Sprint 1 (11/30)

  • The user will be able to successfully boot the program.
  • The user will be able to display all orders for a specified date.

    • Sprint 2 (12/06)

  • The user will be able to add an order to memory.
  • The user will be able to remove an order from memory.
  • The user will be able to edit an order in memory.
  • The user will be able to toggle production/training mode.

    • Sprint 3 (12/12)

  • The user will be able to save orders to a file named by date without replacing an existing file (if it exists).
